## About the Project
This is a desktop-based Accounting Management System developed to efficiently manage financial records, track transactions, and streamline accounting processes. The application features a user-friendly Graphical User Interface (GUI) and is fully integrated with a relational database to ensure data persistence and security.

## Key Features
* **Interactive GUI:** Designed using Java Swing for a seamless and intuitive user experience.
* **Database Integration:** Connected to an SQL database (via JDBC) to securely store and manage accounting data.
* **CRUD Operations:** Full capability to Create, Read, Update, and Delete financial records dynamically.
* **Object-Oriented Design:** Built with solid OOP principles ensuring clean, modular, and maintainable code.

## Technologies & Tools Used
* **Programming Language:** Java
* **UI Framework:** Java Swing
* **Database:** SQL (Relational Database Management System)
* **Core Concepts:** Object-Oriented Programming (OOP), Data Management, UI/UX Basics

Through this project, I strengthened my understanding of desktop application development, relational database architectures, and bridging the gap between back-end logic and front-end interfaces.
